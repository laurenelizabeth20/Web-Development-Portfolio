Week 4 Reading Notes

Object-Oriented Programming

Create a dice object:

const Dice = function(sides=6){ this.sides = sides; this.roll = function() { return Math.floor(this.sides * Math.random() + 1) } }

'this' is used to represent the object that will be returned by the constructor function.

'sides' is a property of the dice object.

'roll' is a method of the dice object.

Instance of dice:

const redDice = new Dice();

'new' is an operator to create a new instance of an object. Parentheses after object name not required.

Checking to see if redDice is an instance of Dice:

redDice instanceof Dice

Forms

The form object has a method called elements. This form has two controls which are the input and button elements:

const [input,button] = form.elements;

Setting the action of a form:

form.action = '/an/other.url'
